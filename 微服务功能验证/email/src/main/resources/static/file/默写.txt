Redis默写
如Linux安装
make install PREFIX=/usr/local/redis
/usr/local/redis/bin/redis-server redis.conf 开启连接
/usr/local/redis/bin/redis-cli 开启操作
	shutdown 关闭连接
	ping 测试连接，返回pong
	echo 信息 返回信息表示连接成功

	字符串类型
	set key value
	mset key value key value...
	get key
	mget key key key
	del key1 key2
	
	哈希类型
	hset key field value
	hget key field
	hmset key field1 value1 field2 value2...
	hget key field1 field..
	hdel key
	del key1 key2..


keys * 所有key
keys a* a开头的key
hkeys key 所有属性名
